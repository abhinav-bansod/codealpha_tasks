# CodeAlpha Python Internship Projects

This repository contains Python projects completed as part of the CodeAlpha Python Programming Internship.

## Projects Included

### 1. Basic Chatbot
A simple rule-based chatbot that responds to predefined user inputs using conditional statements and loops.

Run:
python Chatbot/chatbot.py

### 2. Hangman Game
A text-based Hangman game where the player guesses letters with a limited number of attempts.

Run:
python Hangman/hangman.py

### 3. Stock Portfolio Tracker
A simple program that calculates total investment using predefined stock prices.

Run:
python PortfolioTracker/portfolio.py
